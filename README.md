# Touckey

Touckey 是一个 Android 应用，可将手机变成面向桌面系统的蓝牙键盘与触摸板。

## 开发

- 使用 Android Studio 打开项目根目录
- 确保本机已安装 Android SDK，并为项目提供 `local.properties` 或 `ANDROID_HOME`
- 可先运行 `./gradlew help` 验证 Gradle Wrapper
